# Security Policy

Supported versions and Python versions: **replace before adoption**

Report vulnerabilities through the repository's private security-advisory
form. Do not disclose exploit details or secrets in a public issue.

Acknowledgement target: five business days. Status-update target: ten business
days. Confirmed critical and high vulnerabilities block release unless a
documented, approved, time-bounded exception exists.

Release signing/attestation status: **replace before adoption**
